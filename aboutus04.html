招贤纳士

以下是各个空缺岗位具体职责与要求，有意者请详阅，并发简历、相应作品及个人阐述至：hr@puxinjinfu.com

网站推广专员

岗位职责（工作内容和主要职责）： 
1、负责公司主营业务、品牌线上的宣传推广，制定执行推广方案，跟踪和分析推广效果，进行持续优化和调整，提升网站流量和知名度，促进运营目标； 
2、熟悉各种搜索引擎，其他新兴网络推广媒介，负责网站关键字的排名，推广账户管理； 
3、跟踪、搜集与公司相关的网络市场信息，针对存在的问题提出解决方案； 
4、有良好的文字功底，撰写软文，负责网站链接推广； 
5、向主管经理日常工作汇报。 

任职要求： 
1、工作积极主动，责任心强，善于思考学习，具备较高敬业精神和执行力及团队合作精神，有较强的语言表达沟通能力； 
2、2年以上的网站推广经验； 
3、 擅长各类网络推广方式、宣传文案的策划与写作、善于网上沟通与外联，熟悉网站内容编辑工作，能进行基本的网页编辑、图形处理，对网络营销有深入专研； 
4、 有推广及运营过中大型网络营销类网站者优先。

网站前端

岗位职责（工作内容和主要职责）： 
1、负责网站基础组件开发，前端页面制作，以及前端与业务层交互设计、开发； 
2、优化及提高前端代码的质量，提升前端网站加载速度，包括服务端与客户端的整体用户体验； 
3、有分析、调试、优化和解决问题的能力，能够独立完成项目开发。 
岗位要求： 
1、2年以上工作经验，精通HTML5/CSS3/Javascript/js等前端技术，了解Node、BootStrap、Jade、backbone等，能够独立完成Web页面设计； 
2、深刻理解Web标准，了解不同浏览器之间的差异，写出的前端代码具有良好的兼容性，熟练使用JQuery?js等JS前端开发框架及响应式开发，多种终端访问； 
3、具有良好的创意设计能力，有较强的审美观念及色彩搭配能力，能够驾驭网站的整体风格设计，具有把握不用风格页面的能力； 
4、熟悉换联网产品的特性，有较强的产品意识，对用户体验和交互设计有较深的理解； 
5、能和编程团队进行良好的合作，适应一定工作压力

美工

岗位职责（工作内容和主要职责）： 
1、 负责公司官方网站的整体设计及实现，按产品及客户群体优化网站版面结构； 
2、 按要求对单个产品设计对应的线上，线下宣传材料；包括但不限于产品形象定位设计、广告海报，视觉营销等； 
3、 对接相关广告公司相关接口完成公司对宣传材料的设计和优化； 
4、 不断对网站客户美学体验提供建议和优化。 
岗位要求： 
1、相关专业本科及以上学历，两年以上网页设计、美工工作经验； 
2、有丰富的互联网相关设计经验，对目前各新型网站的设计风格有敏锐的嗅觉力； 
3、精通Flash/Photoshop/Coreldraw/ Dreamweaver/Illustrator等相关设计工具； 
4、熟悉W3C标准，熟悉DIV+CSS方式页面设计，熟练掌握HTML/CSS/JAVASCRIPT/ Ajax/jQuery等页面技术及工具的使用； 
5、熟悉网站制作流程，能够与程序开发人员很好的协作； 
6、具有较强的美术功底，擅长创意，有良好的沟通、协作能力、诚实守信；

客服

岗位职责（工作内容和主要职责）： 
1. 接听客服电话，利用即时通讯工具，解答顾客关于网站的咨询、疑问，增加客户体验 
2. 与其他部门保持良好的沟通，及时提交和反馈各项工作信息。 
3. 为客户提供优质温馨的回访服务，提高顾客满意度。 
任职要求： 
1. 本科以上 
2. 声音甜美，普通话标准，沟通表达能力佳； 
3. 良好的执行力和团队合作精神； 
4. 熟悉客户服务的业务模式，有相关工作经验的优先。 
5. 有过股票投资经验或行业经验的优先

编辑

岗位职责（工作内容和主要职责）： 
1. 负责网站信息内容的搜集、把关、规范、整合和编辑，并更新上线； 
2. 撰写和发布产品文案和稿件； 
3. 协调公司内其他部门，完成方案策划、内容制作等相关工作；配合技术、市场等其他部门的工作； 
4. 撰写软文推广网站、微信公共号、微博、QQ群等相关新媒体的内容更新。 
任职要求： 
1. 文案功底扎实，文笔流畅，具有处理新闻标题和策划专题的能力 
2. 熟悉股票投资，编辑过相关股票投资分析文章的优先 
3. 良好的执行力和团队合作精神； 
4. 思想创新，喜欢突破与超越；

渠道销售

岗位职责（工作内容和主要职责）： 
1. 根据渠道市场推广策略，负责实施渠道的市场推广和销售工作； 
2. 完成区域渠道拓展任务，维系渠道关系，实施渠道管理； 
3. 完成公司下达的销售任务； 
4. 及时向渠道传达公司最新销售政策和市场动态信息，收集渠道综合信息和竞争对手信息。 
任职要求： 
1. 大学本科以上学历 
2. 具有两年以上渠道销售工作经验，熟悉证券行业； 
3. 具有出色的市场开拓和谈判能力； 
4. 具有较强的抗压能力和良好的团队合作意

财务总监

岗位职责（工作内容和主要职责）： 
1、负责组织公司全盘账务处理:包括会计核算 填制会计凭证 登记会计账薄、月度结账、编制会计报表等； 
2、负责编制公司会计，统计 管理报表； 
3、负责审核线上与线下各项收支预算，加强成本费用的核算与控制； 
4、负责向主管税务机关：国税地税定期纳税申报； 
5、负责与其他政府主管部门的报表报送，备案和年审； 
6、协助财务负责人拟定并完善相关财务，会计制度，并协调执行； 
7、协助财务负责人与相关业务部门合作，参与业务流程与内部控制流程的拟定，保障公司财务运营与会计核算的效率及有效性； 
8、协助组织相关财税，会计等档案资料的管理； 
9、接受上级及其他工作安排； 
任职要求： 
1、学历和技能：财务会计 审计金融等相关专业；专科及以上学历，能熟练运用WORD,EXCEL等办公软件； 
2、工作经验：5年以上财务工作经验，其中具金融或金融类企业经验优先，熟悉金融政策法规及会计法规，会计准则，财税政策法规； 
3、具有良好职业道德，责任心及自我管理能力强，良好团队协作精神。
